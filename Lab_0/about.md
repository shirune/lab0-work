Git — это распределенная система контроля версий. Она позволяет:
* Отслеживать все изменения в файлах проекта и при необходимости возвращаться к любой предыдущей версии.
* Параллельно работать над разными задачами, создавая ветки (branches), и затем объединять результаты.
* Надежно хранить полную историю разработки на локальном компьютере каждого участника, что позволяет работать автономно

**Для чего будет использоваться GitHub Classroom?**
GitHub Classroom — это инструмент для автоматизации учебных курсов. Преподаватели используют его для:
* Создания и распространения индивидуальных или групповых заданий среди студентов
* Автоматической проверки выполненных работ с помощью настраиваемых тестов (autograding)
* Отслеживания прогресса каждого студента и оставления обратной связи прямо в коде через pull request
